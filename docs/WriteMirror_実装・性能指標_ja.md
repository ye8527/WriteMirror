# WriteMirror 0.6.0 実装・性能指標

更新日：2026年7月19日

## 実装規模

生成物、`bin`、`obj`、外部ライブラリを除いたチーム管理下のソースを物理行数で集計しました。空行とコメントを含むため、論理行数ではありません。

| 区分 | ファイル数 | 物理LOC |
|---|---:|---:|
| `src` | 34 | 4,865 |
| `tests` | 11 | 983 |
| `experiments` | 4 | 484 |
| 合計 | 49 | 6,332 |

| 言語 | 物理LOC |
|---|---:|
| C# | 5,362 |
| XAML | 512 |
| Python | 458 |

上表はアプリ、テスト、学習実験のコードです。これとは別に、インストール、実機QA、公開監査、文書生成、デモ録画を再現する支援コードが13ファイル、1,719行あります（PowerShell 850、Python 859、CMD 10）。両者を合わせたチーム作成コードは62ファイル、8,051行です。いずれも空行とコメントを含む物理行数で、生成物、`bin`、`obj`、学習データ、外部ライブラリは含みません。

主要部分にはWPF、Core/Infrastructure、WinUIの先行試作を含みます。Windows日本語手書き認識はWPFからOS APIを直接呼び出します。

## 主要言語・フレームワーク

- C# / .NET 9 / WPF / Windows Ink
- Windows App SDK 2.1 ML / Windows ML / ONNX Runtime
- Qualcomm QNN Execution Provider / Hexagon NPU
- Python / PyTorch / NumPy / svgpathtools（学習時のみ）
- MSTest（Core単体テスト）

## オフライン性と制約

- 学習済みモデルはアプリへ同梱され、通常推論、筆跡処理、文字候補、音声出力は端末内で完結します。
- 認定QNN実行プロバイダーがない場合、初回の `EnsureReadyAsync` だけWindows側のダウンロードを必要とする場合があります。
- クラウドAI、外部API、利用者アカウント、アナリティクスを使用しません。
- AI値は本人回答の確定後、または本人が観測表示を明示選択した後だけ計算・表示します。
- NPUのない端末または準備失敗時はCPUモードと明示し、NPU実行とは表示しません。
- x64版はビルド済みですが、Intel/AMD実機での最終操作確認は未実施です。
