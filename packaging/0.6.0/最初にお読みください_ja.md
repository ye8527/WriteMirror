# WriteMirror 0.6.0 ARM64 配布パッケージ

このパッケージは、Snapdragon Xシリーズを搭載したWindows 11 ARM64 Surface向けです。Surface Pro 11実機で、インストール、Surface Pen入力、Windows日本語文字候補、日本語音声、Windows ML、Qualcomm Hexagon NPU推論を確認しています。他の端末での無条件の動作保証ではありません。

## インストール

1. ZIPを端末内の通常フォルダーへすべて展開します。ZIP内から直接実行しないでください。
2. `SHA256SUMS_ja.txt`の値を必要に応じて確認します。
3. `Install-WriteMirror.cmd`をダブルクリックします。
4. インストールが完了するとWriteMirrorが起動します。次回からはスタートメニューの「WriteMirror」を選びます。

インストーラーは、同梱した自己署名の公開証明書を現在ユーザーの`TrustedPeople`へ登録します。秘密鍵、パスワード、管理者アカウントは含みません。通常は現在ユーザーの範囲で処理しますが、学校・会社が管理する端末で承認画面やブロックが出た場合は、組織の管理方針に従ってください。管理者情報を他のメンバーへ共有しないでください。

## 初回のAI準備

アプリには量子化ONNX軌跡モデルが同梱されています。対応するQNN実行プロバイダーが端末にない場合、初回起動時にWindows MLの`EnsureReadyAsync`がWindows側で認定コンポーネントを取得するため、インターネット接続と数分程度の待ち時間が必要になる場合があります。準備後のモデル推論、筆跡処理、文字候補、音声は端末内で行い、筆跡をクラウドへ送りません。

画面上部が次のいずれかになるまで待ってから操作してください。

- `Qualcomm Hexagon NPU（QNN）`：NPU実行を確認できた状態
- `CPUモード`：NPUを利用できないが、基本機能とAI推論は利用できる状態
- `AIモデルを利用できません`：基本の書字記録は利用できるが、AI表示は利用できない状態

## 簡単な動作確認

1. 起動時の説明を読み、「つかう」を選びます。
2. 「確認・デモ設定」を開き、「デモデータを使う」を選びます。
3. 「書いたあとの気もち」は「特になし」のまま「書けた・ふり返る」を選びます。
4. この時点で観測候補とAI結果が自動表示されないことを確認します。
5. 「観測を見てみる（任意）」を選びます。
6. 灰色の入力軌跡と紫色のAI再構成、実行プロバイダー、推論時間が表示されることを確認します。

AI再構成は正解見本、採点、診断、能力評価ではありません。学習元はKanjiVGの漢字であり、平仮名・片仮名のAI妥当性は未検証です。文字候補はAI軌跡モデルとは別のWindows機能です。

## Surface Penで使う

- 白い書字領域へSurface Penで書きます。ペンがない場合はマウスでも入力できます。
- 右手・左手は「使う手」で自由に選べます。
- 対象文字は自由入力でき、Windows文字候補は任意機能です。
- 既定の「ひとりで練習（保存しない）」では、セッションをファイルへ保存しません。

## アンインストール

`Uninstall-WriteMirror.cmd`をダブルクリックするか、Windowsの「インストールされているアプリ」からWriteMirrorを削除します。保存を明示したセッションJSONは`%LOCALAPPDATA%\WriteMirror\Sessions`にあり、必要に応じてアプリ内の削除機能を先に使用してください。

## デモ動画

同梱の `WriteMirror_デモ動画_ja_0.6.0.mp4` は、Surface Pro 11上の0.6.0.4を実際に操作した55秒の日本語字幕付き動画です。動画は無音で、説明は字幕として表示されます。
